# effective visual studio

主要介绍`c/c++`下的快捷键。



# 快捷键

需要安装`Visual Assist X`

| 功能                   | 快捷键       | 应用场景                   |
| :--------------------- | :----------- | -------------------------- |
| 跳转到定义             | F12          |                            |
| 头文件和源文件间切换   | Alt+O        |                            |
| 打开文件               | Ctrl+，      |                            |
| 打开头文件             | Ctrl+Shift+G |                            |
| 跳转到某行             | Ctrl+G       |                            |
| 查找符号在哪儿被调用   | Alt+Shift+F  | 要将光标放在要查找的符号上 |
| 查找符号在哪儿定义     | Alt+Shift+S  |                            |
| 查看近期所有打开的文件 | Ctrl+Tab     |                            |
